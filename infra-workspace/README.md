# Infra Workspace

Infrastructure admin workspace with CLIs and Charm tools.

## What's included

- **CLIs**: coder, dokploy, hapi (Hostinger)
- **Charm tools**: gum, glow, vhs, skate
- **Git tools**: LazyGit, LazySql
- **Editor**: Neovim + LazyVim
- **Shell**: zsh, tmux, Starship
- **Runtime**: Bun (for AI agent installation)
- **IDE**: code-server (VS Code in the browser) — always on

## AI Agents (optional, all default off)

Select at workspace creation:

| Agent | Package |
|-------|---------|
| Claude Code | `@anthropic-ai/claude-code` |
| GitHub Copilot CLI | `gh copilot` extension |
| Codex CLI | `@openai/codex` |
| OpenCode | `opencode-ai` |

## Use cases

- Managing Dokploy deployments
- Pushing Coder templates
- Managing Hostinger VPS
- Reviewing repos with LazyGit
- Reading docs with Glow
- Recording terminal sessions with VHS
